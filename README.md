<h1>saipackage 6.x</h1>

基于<a href="https://saithink.top" target="_blank">saiadmin</a>创建的插件安装器，能够方便的管理安装`saiadmin`插件

<h1>安装</h1>

composer环境的安装命令如下

```bash
composer require saithink/saipackage:^6.0
```

本插件必须在`saiadmin`环境下才能使用，请先配置`saiadmin`
